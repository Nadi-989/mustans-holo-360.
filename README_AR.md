# Mustans Holo 360 3D (Flutter)

تطبيق موبايل لمعالجة الصور مع:
- خلفية سوداء للصورة.
- علامة مائية `Mustans_Holo`.
- معاينة دوران 360° بشكل 3D.
- تصدير الدوران كـ `GIF` وكـ `MP4`.
- دقة إخراج عالية مناسبة للعرض الهولوغرامي (حتى 3072x3072).

## المتطلبات
- Flutter SDK
- Android Studio + Android SDK

## التشغيل
```bash
flutter create .
flutter pub get
flutter run
```

## الاستخدام داخل التطبيق
1. `اختيار صورة`
2. `معالجة` (خلفية سوداء + علامة مائية)
3. اختيار الدقة المناسبة (`2048` أو `3072` للهولوغرام)
4. تصدير `GIF` أو `MP4`

## بناء Google Play
```bash
flutter build appbundle --release
```
الملف الناتج:
`build/app/outputs/bundle/release/app-release.aab`

## ملاحظة للهولوغرام
للحصول على نتيجة أفضل في الشاشات/المراوح الهولوغرامية:
- استخدم صورة مربعة أو شبه مربعة.
- اختر دقة `2048` أو `3072`.
- اختر `24 FPS` أو `30 FPS` لحركة أنعم.
